# SynMux
**Advanced opinionated Termux dotfiles focused on UX**

*NOTE: This project is still in it's alpha stages, there wil be inconsistancy and problems. feel free to contribute to the project.*


## Features

- Live Autocompletion for commands.
- Syntax highlighting in command line.
- Pre configured tools for everything.
- Erganomic setup, specialized for Termux.
- Modern and beautiful theming.
- Modernization of the package manager.

## How to install.

Run the following command in your termux '~' directory.
```apt upgrade && pkg update && pkg install git && git clone https://github.com/SyntacticallySilly/dotfiles.git && bash ~/.dotfiles/init.sh```
